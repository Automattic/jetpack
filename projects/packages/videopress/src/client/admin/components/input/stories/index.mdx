## API

### value

- type: `string`

The input value.

### placeholder

- type: `string`

The input placeholder.

### disabled

- type: `boolean`

Whether the input is disabled.

### icon

- type: `ReactNode`

An optional icon.

### onChange

- type: `function`

Callback to be invoked when the input value changes.

### onEnter

- type: `function`

Callback to be invoked when the user presses the Enter key.
