# Buttons

This page contains all variations for RNA buttons. These use the `Button` component from `@wordpress/components` and are styled with our own Jetpack styles.

<Story name="Example button">
	<Button variant="primary" href="https://jetpack.com/" target="_blank">
		Love me some buttons
	</Button>
</Story>

The file you are looking at lives in js-packages/base-styles/stories/button.stories.mdx.
